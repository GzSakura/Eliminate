# Eliminate 配置说明

## 概述

Eliminate模组提供了丰富的配置选项，可以通过ModMenu或直接编辑配置文件进行自定义。所有配置选项都支持中文和英文界面。

## 配置分类

### 1. 通用 (General)

| 选项 | 默认值 | 说明 |
|------|--------|------|
| 启用模组 | true | 启用或禁用Eliminate模组的所有功能 |
| 剔除距离 | 32 | 区块剔除的最大距离（方块），范围：8-256 |
| 调试模式 | false | 在操作栏显示详细的剔除统计信息 |
| 自动优化 | false | 根据系统性能自动调整配置 |

### 2. 剔除 (Culling)

| 选项 | 默认值 | 说明 |
|------|--------|------|
| FOV剔除 | true | 基于视野角度剔除不可见的区块 |
| 激进山体剔除 | true | 在地下时激进地剔除地表区块 |
| 透明度感知 | true | 检测透明方块以避免视觉闪烁 |
| 水平剔除 | true | 剔除水平方向过远的区块 |
| 生物群系感知 | true | 根据生物群系类型调整剔除策略 |
| 末地维度优化 | true | 为末地维度应用特殊优化 |

### 3. 垂直剔除 (Vertical Culling)

| 选项 | 默认值 | 说明 |
|------|--------|------|
| 启用垂直剔除 | true | 根据玩家高度剔除垂直方向过远的区块 |
| 垂直剔除距离 | 32 | 垂直方向的最大剔除距离，范围：8-128 |
| 仅地下垂直剔除 | false | 仅在玩家位于地下时应用垂直剔除 |
| 地表边距 | 8 | 地表附近的剔除边距，范围：0-32 |

### 4. 背面剔除 (Backface Culling)

| 选项 | 默认值 | 说明 |
|------|--------|------|
| 背面剔除 | false | 剔除背对玩家的区块 |
| 激进背面剔除 | false | 使用更激进的背面剔除算法 |
| 背面剔除距离 | 64 | 背面剔除的最大距离，范围：16-256 |

### 5. 实体剔除 (Entity Culling)

| 选项 | 默认值 | 说明 |
|------|--------|------|
| 实体剔除 | false | 剔除视野外的实体 |
| 实体剔除距离 | 128 | 实体剔除的最大距离，范围：32-512 |
| 包含物品 | true | 剔除掉落的物品 |
| 包含粒子 | false | 剔除视野外的粒子效果 |

### 6. 雾效剔除 (Fog Culling)

| 选项 | 默认值 | 说明 |
|------|--------|------|
| 雾效剔除 | true | 在雾效范围内剔除区块 |
| 雾效剔除距离 | 96 | 雾效剔除的最大距离，范围：32-256 |
| 自适应雾效剔除 | true | 根据当前雾效密度动态调整剔除 |

### 7. 性能 (Performance)

| 选项 | 默认值 | 说明 |
|------|--------|------|
| 动态剔除距离 | false | 根据FPS动态调整剔除距离 |
| 目标FPS | 60 | 动态调整的目标帧率，范围：30-300 |
| 最小动态距离 | 16 | 动态剔除的最小距离，范围：8-64 |
| 最大动态距离 | 64 | 动态剔除的最大距离，范围：16-256 |

### 8. 高级 (Advanced)

| 选项 | 默认值 | 说明 |
|------|--------|------|
| 更新速度 | 20 | 配置更新的速度（1-20），值越小更新越快 |
| 与Sodium同步 | false | 与Sodium模组的渲染周期同步 |
| 线程池大小 | -1 | 异步任务的线程池大小（-1为自动检测），范围：-1-32 |
| 预计算区块 | true | 在后台预计算附近区块 |
| 预计算半径 | 8 | 预计算区块的半径，范围：1-32 |

### 9. 缓存 (Cache)

| 选项 | 默认值 | 说明 |
|------|--------|------|
| 启用缓存优化 | true | 使用缓存来提高性能 |
| 最大缓存大小 | 1024 | 缓存的最大条目数，范围：128-8192 |
| 缓存生存时间 | 300 | 缓存条目的生存时间（秒），范围：60-3600 |
| 持久化缓存 | false | 在游戏会话之间保持缓存 |

### 10. 维度 (Dimension)

#### 下界优化

| 选项 | 默认值 | 说明 |
|------|--------|------|
| 下界优化 | true | 为下界维度应用特殊优化 |
| 下界天花板剔除距离 | 128 | 下界天花板剔除的最大距离，范围：64-256 |
| 下界地板剔除 | true | 启用下界地板剔除 |
| 下界地板剔除距离 | 64 | 下界地板剔除的最大距离，范围：32-128 |

#### 末地优化

| 选项 | 默认值 | 说明 |
|------|--------|------|
| 末地优化 | true | 为末地维度应用特殊优化 |
| 末地岛屿剔除距离 | 96 | 末地浮空岛剔除的最大距离，范围：48-256 |
| 末地虚空剔除 | true | 启用末地虚空区域剔除 |
| 末地虚空剔除距离 | 128 | 末地虚空剔除的最大距离，范围：64-256 |

### 11. 调试 (Debug)

| 选项 | 默认值 | 说明 |
|------|--------|------|
| 启用调试HUD | false | 在屏幕上显示调试信息 |
| HUD刷新率 | 20 | 调试HUD的刷新频率，范围：5-60 |
| 显示缓存统计 | true | 在HUD中显示缓存统计信息 |
| 显示线程统计 | true | 在HUD中显示线程池统计信息 |
| 显示剔除明细 | true | 在HUD中显示详细的剔除分类 |

### 12. 性能分析 (Profiling)

| 选项 | 默认值 | 说明 |
|------|--------|------|
| 启用性能分析 | false | 收集性能数据用于优化 |
| 分析采样间隔 | 60 | 性能分析的采样间隔（tick），范围：10-300 |
| 自动优化 | false | 基于性能分析自动调整配置 |

### 13. 兼容性 (Compatibility)

| 选项 | 默认值 | 说明 |
|------|--------|------|
| 兼容模式 | false | 启用与其他模组的兼容性调整 |
| Sodium集成 | true | 启用与Sodium模组的深度集成 |
| Iris集成 | true | 启用与Iris光影模组的集成 |
| OptiFine兼容模式 | false | 启用OptiFine兼容性调整 |

### 14. 实验性 (Experimental)

| 选项 | 默认值 | 说明 |
|------|--------|------|
| 实验性功能 | false | 启用实验性功能（可能不稳定） |
| 实验性异步渲染 | false | 使用实验性的异步渲染技术 |
| 实验性预测剔除 | false | 使用预测算法提前剔除区块 |
| 预测半径 | 32 | 预测剔除的半径，范围：16-128 |

## 配置文件位置

配置文件位于：`config/eliminate.json`

## 推荐配置

### 低配置设备

```json
{
  "cullingDistance": 16,
  "verticalCullingDistance": 16,
  "dynamicCullingDistance": true,
  "targetFps": 30,
  "cacheMaxSize": 512,
  "preCalculateRadius": 4
}
```

### 中等配置设备

```json
{
  "cullingDistance": 32,
  "verticalCullingDistance": 32,
  "dynamicCullingDistance": true,
  "targetFps": 60,
  "cacheMaxSize": 1024,
  "preCalculateRadius": 8
}
```

### 高配置设备

```json
{
  "cullingDistance": 64,
  "verticalCullingDistance": 64,
  "dynamicCullingDistance": false,
  "cacheMaxSize": 2048,
  "preCalculateRadius": 16,
  "experimentalFeaturesEnabled": true
}
```

## 故障排除

### 配置未生效

1. 确保配置文件路径正确
2. 检查配置文件格式是否正确（JSON格式）
3. 重启游戏使配置生效

### 性能未提升

1. 尝试启用"动态剔除距离"
2. 增加"剔除距离"的值
3. 启用"预计算区块"
4. 检查是否与其他优化模组冲突

### 视觉问题

1. 禁用"激进山体剔除"
2. 启用"透明度感知"
3. 降低"剔除距离"的值
